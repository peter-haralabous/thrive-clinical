## HOW TO USE THIS CODEBOOK

This codebook provides the most comprehensive systematic framework for coding qualitative data from provider interviews. Each code includes:
- **Definition:** Clear description of what the code captures
- **When to use:** Guidance on when to apply this code
- **When NOT to use:** Boundary conditions to prevent overlap
- **Example quotes:** 5+ representative quotes from different participants

**Coding Rules:**
1. Codes are mutually exclusive at the subcode level
2. A single quote may be assigned to only ONE subcode
3. If a quote spans multiple concepts, code to the PRIMARY theme
4. Use participant identifiers (e.g., VW-PR08) for traceability

# MAIN CODE 1: WORKFLOW EFFICIENCY & TIME MANAGEMENT

**Definition:** This main code captures all aspects of time spent on administrative tasks, preparation activities, workflow inefficiencies, and systemic barriers that detract from direct patient care. It includes both the actual time burden and the impact on provider capacity and job satisfaction.

**When to use:** Apply when providers discuss time spent on tasks, workflow bottlenecks, efficiency challenges, administrative burden, or system-level process inefficiencies.

**When NOT to use:** Do not use for discussions about clinical decision-making time, patient interaction time, or time spent on actual care delivery (unless explicitly framed as burdensome).

1. Read the entire transcript segment
2. Identify the PRIMARY theme (only one subcode per segment)
3. If multiple themes are present, code to the most dominant/explicit theme
4. Use participant ID format: NAME-TRANSCRIPT# (e.g., VW-PR08)

- Update codebook definitions as needed
